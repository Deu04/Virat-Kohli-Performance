# Virat Kohli Performance Analysis Dashboard

The Virat Kohli Performance Analysis Dashboard is an interactive Business Intelligence project developed to analyze and visualize the cricket career statistics of Virat Kohli across ODI and T20 formats. The dashboard provides comprehensive insights into his batting performance, consistency, achievements, and career progression through interactive charts and visualizations.

The project was developed using Power BI and cricket statistics datasets to transform raw performance data into meaningful analytical insights for cricket enthusiasts, analysts, and sports data learners.

## 🎯 Project Objectives

* Analyze Virat Kohli's ODI and T20 career statistics.
* Visualize batting performance across formats.
* Track runs, centuries, half-centuries, and strike rates.
* Evaluate consistency and career progression.
* Provide interactive insights through dynamic dashboards.
* Demonstrate sports analytics using Business Intelligence tools.
